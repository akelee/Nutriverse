# Nutriverse 
nutriverse.app  -- a platform to provide a convenient means of meal planning based on a user's profile and nutritional needs.
# Project Documentation

# To run the program:
1. Clone the nutriverse project
2. Run npm install 
3. Run node app.js
4. Run npm start
